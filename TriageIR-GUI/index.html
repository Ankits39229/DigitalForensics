Welcome to TriageIR

TriageIR is a professional digital forensics triage tool designed for incident responders and forensic analysts.


Features:

  • Rapid system information collection
  • Running process enumeration
  • Network connection analysis
  • Persistence mechanism detection
  • Windows event log collection
  • Professional JSON output format

Getting Started:

  1. Click "Start Full Scan" for complete forensic collection
  2. Use "Quick Scan" for faster basic collection
  3. View results in the tabs above
  4. Export results for further analysis
